# React Vite Redux CRUD Dashboard

## Objective
To create a React application using **Vite** that fetches data from a public API, stores it in **Redux Toolkit**, and performs **CRUD operations**.

---

## Technologies Used
- React (Vite)
- Redux Toolkit
- React Redux
- Axios
- JavaScript

---

## API Used
https://jsonplaceholder.typicode.com/users

---

## Features Implemented
- Fetch data from GET API
- Store data in Redux store
- Create, Read, Update, Delete (CRUD)
- Display data in dashboard table
- Loading and error handling

---

## Project Structure
src/
├─ app/store.js
├─ features/users/
├─ components/UserTable.jsx
├─ App.jsx
└─ main.jsx

---

## Steps to Run the Project
1. Clone the repository  
git clone <repo-url>

2. Install dependencies  
npm install

3. Run the project  
npm run dev

4. Open browser:  
http://localhost:5173

---

## CRUD Summary
| Operation | Description |
|--------|------------|
| Create | Add user |
| Read | Fetch & display users |
| Update | Edit user |
